1351. 阅读程序写结果 (Standard IO)

Time Limits: 1000 ms Memory Limits: 65536 KB

Description

NOIP2009普及组初赛阅读程序第三题如下：
const c = 2009;
var
n, p, s, i, j, t: integer;
begin
read(n, p);
s := 0;
t := 1;
for i := 1 to n do
begin
t := t * p mod c;
for j := 1 to i do
s := (s + t) mod c;
end;
writeln(s);
end.
输入：11 2
输出：______
这题很简单，死做都能做出来，但如果扩大N的范围，你还能做吗？

Input

输入N(1<=N<=10^1000),P(1<=P<=1000),之间用一个空格隔开。

Output

输出结果。

Sample Input 

11 2

Sample Output 

782

Data Constraint

Hint

【数据规模】
50%的N<=1,000,000
