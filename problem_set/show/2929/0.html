6406. 【NOIP2019模拟11.05】小 D 与原题
(File IO): input: problem.in output: problem.out

Time Limits: 1000 ms Memory Limits: 528400 KB

Description

Input

一个正整数 n，保证 n 是偶数。

Output

输出 n-1 行，每行 n 个正整数。
第 i 行的第 2k-1 和第 2k 个整数表示小 D 在第 i 次考试中会将这两道题拼成一道新题。

Sample Input 

2

Sample Output 

1 2
 

Data Constraint

对于 30% 的数据，n≤10。
对于 100% 的数据，n≤1000。
