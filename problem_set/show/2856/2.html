3427. 归途与征程 (Standard IO)

Time Limits: 1000 ms Memory Limits: 131072 KB

Description

Input

第一行为字符串A。

第二行为字符串B。

Output

输出在B的所有循环同构串中，有多少个能够与A匹配。

Sample Input 

输入1：

aaaa

aaaa

输入2：

a*a

aaaaaa

输入3：

*a*b*c*

abacabadabacaba

Sample Output 

输出1：

4

输出2：

6

输出3：

15

Data Constraint

对于30%的数据，M<=20；

对于80%的测试点，M<=200；

对于100%的测试点，1<=N<=100,1<=M<=100000。
