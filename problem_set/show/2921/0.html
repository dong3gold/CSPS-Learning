6280. 【noip提高组模拟2】敏感词
(File IO): input: replace.in output: replace.out

Time Limits: 1000 ms Memory Limits: 524288 KB

Description

有一个专制的国家，对于网络言论执行着严格的审查。
最近"ab"成为了这个国家的敏感词，如果说出这个词就会被人间蒸发。
所有的网民都会在发言之前仔细自我审查，并且达成了默契，把所有的"ab"替换成"bba"。
然而这样并不能骗过审查，比如"aab"会被替换成"abba"，然后就会被【数据已删除】。
于是网民们会重复替换下去，直到可以躲过审查。
你的任务是在"bba"也被河蟹之前尽快帮网民们计算出需要的替换次数，为这个国家言论自由情况的改善做出一份努力。
可悲的是替换次数会是一个天文数字，所以你只需要算出答案取模(10^9+7)的结果即可。

Input

一行一个串s，只包含a和b两种字符

Output

一行一个数，表示替换次数取模(10^9+7)的结果

Sample Input 

样例输入0
aab
样例输入1
aabaabaabaabaabaabaab

Sample Output 

样例输出0
3
样例输出1
21837

Data Constraint

设n为串长即|s|
对于30%的数据，n<=10
对于另10%的数据，s串为1个a和(n-1)个b组成（如abb...bbb）
对于另20%的数据，s串中不存在"ba"子串
对于100%的数据，n<=500000
