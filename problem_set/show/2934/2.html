3850. Fibonacci进制(fib) (Standard IO)

Time Limits: 1000 ms Memory Limits: 262144 KB

Description

Input

第一行一个整数N，表示统计范围是自然数的Fibonacci 表示的前N个数字。

Output

一个数，前N个数字中1的个数。

Sample Input 

21

Sample Output 

10
【样例解释】
前21个数字为110100101100010011010，共有10个1。

Data Constraint

30%的数据N≤1000；
50%的数据N≤10^6；
100%的数据N≤10^15。
