3803. 数列 (Standard IO)

Time Limits: 4000 ms Memory Limits: 262144 KB

Description

吾妻玲二接到Scythe Master的指令去暗杀反抗他们的黑社会组织头目，他做了详细的调查，发现只要可以进入头目的卧室便可以轻松射杀头目，但是卧室的门上设置了强力的电子锁，只有输入正确的密码才能进入卧室。玲二已经知道了密码的构成规则，即有一个数列a，他需要求出一个等长的数列b满足
最小且数列b中的数两两的最大公约数为1，但是这个问题他难以解决，为了尽快完成任务，他只好求助于擅长编程的你来帮他得到密码。

Input

第1行： 1个整数N表示数列a的长度。
第2行：N个整数分别表示数列a的每一项

Output

输出一行N个数表示数列b，每两个数之间用空格隔开，如果有多个数列b满足要求，输出任意一个即可。

Sample Input 

5
1 6 4 2 8

Sample Output 

1 5 3 1 8

Data Constraint

60%     1<=N<=10
100%    1<=N<=100   1<=Ai<=30
