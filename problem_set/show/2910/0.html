3801. 骰子 (Standard IO)

Time Limits: 1000 ms Memory Limits: 262144 KB

Description

久城一弥参加数学考试不幸挂科了，苦恼的他只好请求维多利亚帮他补习数学，维多利亚给他出了一道题，已知一个m面的骰子，第i面(1<=i<=m)上写着数字i，每个面朝上的概率都是1/m,现在扔这个骰子n次，问期望最大数字是多少？久城一弥苦思冥想仍然得不出答案，但他又不想被维多利亚鄙视，无奈他只好请求你写个程序帮他计算一下答案了。

Input

一行包含两个数m和n，分别表示骰子的面数和扔的次数

Output

输出一个数表示期望最大数字，你的答案和标准答案的误差在0.0001内即视为正确。

Sample Input 

2 2

Sample Output 

1.750000000000
【样例解释】
第一次扔1，第二次扔1，最大数为1
第一次扔1，第二次扔2，最大数为2
第一次扔2，第二次扔1，最大数为2
第一次扔2，第二次扔2，最大数为2期望最大数为(1+2+2+2)/4=1.750000000

Data Constraint

30%  1<=n,m<=10
100%  1<=n,m<=10^5
