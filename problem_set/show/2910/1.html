3802. 排列 (Standard IO)

Time Limits: 4000 ms Memory Limits: 262144 KB

Description

“魔王”向京介的养父发了封恐吓信，意图杀害京介同父异母的妹妹花音，同时留下了讯息，即给你一个1到n的排列，问你是否存在一对数a,b(1<=a,b<=n,a<>b)满足a+b为偶数且(a+b)/2在a和b之间，但是由于排列太过庞大，京介无法破译出这个讯息，连号称天才的宇佐美春也无可奈何，他们只好请求你写个程序来破译讯息，如果存在这样的a和b输出YES，否则输出NO。

Input

本题多测，第一行读入一个数cas表示测试组数
之后每一组的第一行一个整数N，表示排列的大小，第二行N个整数表示这个排列

Output

输出共cas行，对于每组数据如果存在这样一对a,b输出YES否则输出NO

Sample Input 

2
5
1 5 2 4 3
4
1 3 4 2

Sample Output 

YES
NO
【样例解释】
在第一组测试数据中，(1+3)/2=2,2这个数的位置在1这个数的位置和3这个数位置的中间，a=5,b=3也满足要求，所以存在这样一对数。
而第二组数据中则没有这样的a和b。
 

Data Constraint

70%    1<=N<=1000
100%   1<=N<=300000        1<=cas<=10
