6275. 【NOIP提高组模拟1】小L的数列
(File IO): input: seq.in output: seq.out

Time Limits: 1500 ms Memory Limits: 524288 KB

Description

Input

一行两个整数n和k。

之后1行k个正整数b1...bk。

之后1行k个正整数f1...fk。

Output

输出一个整数表示fn

Sample Input 

【样例输入1】

5 4

1 2 3 4

4 3 2 1

【样例输入2】

100000 4

1 2 3 4

12 23 34 45

 

Sample Output 

【样例输出1】

27648

样例输出2】

33508797
 

Data Constraint

对于30%的数据，n≤10000.

对于另外20%的数据，bi=1,n≤1000000.

对于另外20%的数据，f[1]...f[k-1]=1.

对于另外20%的数据，k≤30.

对于100%的数据，1≤k≤200,1≤n≤40000000,1≤bi,fi≤998244352.

Hint

样例解释：1*2*2*3*3*3*4*4*4*4=27648
