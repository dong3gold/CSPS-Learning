3738. 理想城市(city)
(File IO): input: city.in output: city.out

Time Limits: 3000 ms Memory Limits: 262144 KB

Description



Input

第1 行为一个正整数N，为理想城区块的数目。

第2 行到第N+1 行，每行有两个非负整数。第i+2 行为第i 个区块的坐标vi = (xi， yi)。

Output

输出仅一行一个正整数，为S 的值。由于S 的值可能较大，你只需输出S mod 10^9。

Sample Input 

11

2 5

2 6

3 3

3 6

4 3

4 4

4 5

4 6

5 3

5 4

5 6

Sample Output 

174
