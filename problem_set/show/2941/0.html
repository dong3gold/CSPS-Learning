6415. difference
(File IO): input: difference.in output: difference.out

Time Limits: 1000 ms Memory Limits: 65000 KB

Description

Input

第一行一个正整数 表示排列序列的长度,n<=10^6
第二行 个正整数依次为排列序列中的元素

Output

以空格分隔的两个整数，第一个表示向前滚动的次数，第二个表示最小的 d

Sample Input 

样例输入1
3
1 2 3 


样例输入2
3
2 3 1



样例输入3
3
3 2 1 

Sample Output 

样例输出1
0 0 


样例输出2
2 0


样例输出3
1 2

Data Constraint

n ≤ 1000000
ai ≤ 1000000
