6416. 难题(hard)
(File IO): input: hard.in output: hard.out

Time Limits: 1000 ms Memory Limits: 528400 KB

Description

这个难题是这样的：给定一个数列x ，计算 l​ 到 r的所有正整数中，有多少个恰好是x 中某一项的倍数。

Input

多组数据。
第一行，一个正整数T，表示数据组数。
每组数据中：
第一行，一个正整数，表 示 x 的长度。
第二行， n个正整数，表示 x 的每一项。
第三行，两个正整数，表示 l 和 r 。

Output

每组数据一行，表示要求的答案

Sample Input 

2
2
2 3
1 6
2
2 4
1 6

Sample Output 

3
2

【样例解释1】
第一组数据：符号条件的数是2,3,4。注意6同时是2和3的倍数，所以6并不符合条件。
第二组数据：2,6
