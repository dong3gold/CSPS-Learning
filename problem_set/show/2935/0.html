3828. 三角形计数 (Standard IO)

Time Limits: 3000 ms Memory Limits: 8192 KB

Description

给定n个点的坐标(0<=xi,yi<=10000)求选出任意三个点能组成的三角形的总面积。

Input

第一行n表示点数。接下来每行两个数xi,yi表示点的坐标。

Output

一行一个浮点数保留一位小数表示面积和。

Sample Input 

5
0 0
1 2
0 2
1 0
1 1

Sample Output 

7.0

Data Constraint

Hint
