3830. 字符消除 (Standard IO)

Time Limits: 2000 ms Memory Limits: 131072 KB

Description

规则是这样的：每次生牛哥可以选择一个字符串，然后生牛哥选择一个正整数t，系统会给生牛哥一个长度为t的圆环。然后生牛哥在圆环上选择一个起点，从字符串第一个字母开始将这个字符串按顺时针在圆环上放置字母，直到填满为止。即一旦碰到已经放过的位置就不能再放。

接着系统会再给生牛哥一个长度为t的圆环，然后生牛哥选择一个起点从最后一个字母开始倒序按逆时针方向放置字母直到填满为止。
但是生牛哥不能将同一个字母同时放到两个环里面，所以生牛哥选择的t不能超过字符串长度的一半。
接下来系统会消除生牛哥选择的字符串并判断这两个圆环是否相同，如果相同那么生牛哥获得t分，否则生牛哥获得0分。当然假如生牛哥无法选择合法的t那么他只能得到0分。
现在生牛哥有很多个字符串可供选择，他想知道消除每个串的最大得分。

Input

第一行n表示数据组数。
接下来每行一个字符串。（只包括小写字母）

Output

每组数据输出一个数代表最高得分。

Sample Input 

1
ababbabababbaab

Sample Output 

6
