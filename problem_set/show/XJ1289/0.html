友好数对 (Standard IO)

Time Limits: 2000 ms  Memory Limits: 262144 KB

Description
如果一个数a能由一个数b旋转得到，那么我们称为友好数对，如12345和45123为友好数对，12345和54321不为友好数对。给出两个正整数L，R，求有多少友好数对,满足L<=a

Input
第一行一个整数T，表示数据组数，每组数据两个正整数L,R。

Output
对于每组数据，输出一个整数表示答案。

Sample Input

 
输入1：
4
1 9
10 40
100 500
1111 2222

Sample Output

 
输出1：
0
3
156
287

Data Constraint
对于30%的数据满足L,R<=1000
对于100%的数据满足L,R<=2000000，T<=30,L,R位数相同。
