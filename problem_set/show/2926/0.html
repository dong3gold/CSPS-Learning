1219. Num (Standard IO)

Time Limits: 1000 ms Memory Limits: 131072 KB

Description


Input

输入文件仅包括一行，一个整数N(2<=N<=2^31-1)

Output

输出文件仅包括一行，一个整数，表示F(n)

Sample Input 

4

Sample Output 

4

Data Constraint

Hint

对于40%的数据,1≤N≤10^7
对于60%的数据,1≤N≤10^8
对于100%的数据1≤N≤2^31-1
