6273. 2019.8.4【NOIP提高组A】欠钱 (money)
(File IO): input:money.in output:money.out

Time Limits: 1000 ms  Memory Limits: 524288 KB  Detailed Limits 

Description

Input
第一行两个整数 n 和 m，表示有 n 只企鹅，m 个操作。
接下来 m 行，有两种可能的格式：
- 0 a b c：修改操作，企鹅 a 向企鹅 b 借了 c 元钱。
- 1 a b：查询操作，询问假如 a 有了 +∞ 元钱，企鹅 b 会净收入多少钱。
本题强制在线，也就是说：对于每个操作输入的变量 a, b, c（如果没有c，那就只有 a, b）
都不是实际的 a, b, c，想获得实际的 a, b, c 应当经过以下操作：

a = (a + lastans) % n + 1;
b = (b + lastans) % n + 1;
c = (c + lastans) % n + 1;

其中，lastans 是上一次询问的答案。如果没有上一次询问，lastans 为0。

Output
对每个询问操作，输出一行一个数表示答案。

Sample Input

 
5 9
0 1 2 1
0 0 1 2
1 0 1
1 2 4
0 2 1 1
1 2 0
0 3 1 0
1 4 2
1 3 4

Sample Output

 
3
2
0
1
0
